# Visão Geral

Ao fazer qualquer tipo de hacking, é uma **boa prática manter sua máquina pessoal separada da máquina de hacking**. Pense assim: se algo der errado — e eventualmente dará — você não quer perder seus arquivos pessoais ou comprometer sua conta principal.

A solução mais prática é usar uma **máquina virtual** (ou *VM*, de *Virtual Machine*). Uma VM é como um computador completo rodando "dentro" do seu computador real. Ela é isolada: o que acontece lá dentro não afeta o sistema principal.

Nesta aula, vamos configurar uma VM com Windows para usar em todos os exercícios práticos deste curso.

> **Atenção:** Se algum jogo específico não rodar bem na VM (por causa das limitações de hardware virtualizado), uma alternativa é criar uma **partição** no seu HD — uma segunda instalação do Windows dedicada exclusivamente ao hacking.

# VirtualBox

Para essas aulas, usaremos o **VirtualBox** — um software gratuito e de código aberto para gerenciar máquinas virtuais. Você pode baixá-lo em [virtualbox.org](https://www.virtualbox.org/wiki/Downloads).

> **Em 2026**, o VirtualBox continua sendo a opção mais acessível e gratuita. Alternativas como **VMware Workstation Player** também são opções válidas e às vezes oferecem melhor desempenho com jogos 3D.

# A Máquina Virtual

Precisamos de uma VM com Windows, já que a maioria dos jogos é lançada para Windows. A Microsoft disponibiliza imagens de avaliação gratuitas (e legais!) do Windows 10 e 11.

Você pode baixar as imagens aqui:
- [Windows 10 Enterprise (Avaliação)](https://www.microsoft.com/en-us/evalcenter/evaluate-windows-10-enterprise)
- [VMs prontas para desenvolvedores](https://developer.microsoft.com/en-us/windows/downloads/virtual-machines/)

> As imagens têm cerca de **7 GB** — prepare-se para uma espera no download.

Após baixar, extraia o arquivo `.OVF` do pacote. O VirtualBox usa os formatos **OVA** e **OVF** para importar VMs prontas — são como "snapshots" de uma máquina já configurada.

Abra o VirtualBox e importe o arquivo OVF baixado (deixe todas as opções no padrão):

![A Opção de Importar no VirtualBox](/assets/images/1/4/import.png)

Após importar, inicie a VM. O Windows irá abrir e solicitar login. A senha do usuário é **`Passw0rd!`** (sem as aspas).

# Ferramentas

Vamos usar um script do **Boxstarter** para instalar automaticamente as ferramentas de game hacking. O Boxstarter é um utilitário que automatiza instalações no Windows via linha de comando.

Dentro da sua VM, abra o **PowerShell como administrador** e execute os dois comandos abaixo:

```powershell
. { iwr -useb https://boxstarter.org/bootstrapper.ps1 } | iex; Get-Boxstarter -Force
```

```powershell
Install-BoxstarterPackage -PackageName https://raw.githubusercontent.com/GameHackingAcademy/vmsetup/master/vmsetup.txt -DisableReboots
```

O primeiro instala o Boxstarter. O segundo executa nosso script de setup, que instala três programas essenciais:

1. **Cheat Engine** — scanner de memória (para encontrar valores na RAM do jogo)
2. **x64dbg** — debugger (para visualizar e modificar o código do jogo em tempo real)
3. **Chocolatey** — gerenciador de pacotes (para instalar outros programas facilmente)

> **Por que x64dbg?** Em versões anteriores deste curso, usávamos OllyDbg. Mas o OllyDbg está **descontinuado** há anos — não recebe mais atualizações e não suporta aplicações de 64 bits adequadamente. O **x64dbg** é seu substituto moderno: gratuito, de código aberto, ativamente mantido, e suporta tanto 32 quanto 64 bits.

Conforme você descobrir novas ferramentas úteis, adicione-as ao seu próprio script de setup. Assim, se precisar recriar a VM, tudo já estará pronto.

# Fazendo Backup da VM

Com tudo instalado, vamos criar uma cópia da nossa VM configurada. Assim, se ela ficar corrompida (ou a avaliação do Windows expirar), é só reimportar.

Desligue a VM completamente e escolha a opção **Exportar para OCI**. Deixe os padrões e inicie a exportação:

![A Opção de Exportar no VirtualBox](/assets/images/1/4/export.png)

Isso cria um arquivo **OVA** — similar ao OVF que baixamos inicialmente. Guarde esse arquivo em algum lugar seguro. Se precisar recriar o ambiente, basta importar esse OVA: todas as ferramentas já estarão instaladas.
